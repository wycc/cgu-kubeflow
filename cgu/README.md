Installation procesure
# create venv
```
python3 -mvenv cgu
. cgu/bin/activate
```

# install sshscript
```
git clone https://github.com/iapyeh/sshscript.git
pip3 install sshscript
```

# change the password
Change the password in the collect.py to the valid one

# test run
```
# sshscript collect.py
```
